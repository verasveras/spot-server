People's Favorite Cities API Demo

This is a demo of the actions you do with the People's Favorite Cities API.

You can do the following with this API.

  • GET request to /api/people - Gives you all the people in the database.
  • POST request to /api/people - Lets you put a new person in the database.
  • GET request to /api/people/1 - Gives you info about a person in the database with id 1.
  • PUT request to /api/people/1 - Lets you update info about a person in the database with id 1.
  • DELETE request to /api/people/1 - Lets you delete a person in the database with id 1.
